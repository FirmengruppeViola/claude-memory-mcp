# Contributing to Memory Buddy

First off: thanks for being here.

This project was built at 6am by someone who just wanted their AI to remember them. If you're here, you probably get it.

---

## How to Contribute

### Found a Bug?

1. Open an issue
2. Describe what happened
3. We'll figure it out together

### Have an Idea?

1. Open an issue first (let's discuss)
2. Or just make a PR if you're confident
3. We're open to anything that makes this better

### Want to Improve the Code?

1. Fork the repo
2. Make your changes
3. Run `npm test`
4. Submit a PR

### Just Want to Lurk?

Also valid. Star the repo. Come back later. No pressure.

---

## Code Style

- TypeScript
- Keep it simple
- Comments where it's not obvious
- Tests for new features

---

## The Vibe

This isn't a corporate project. It's a gift to the community.

Be kind. Be helpful. Remember we're all nerds here.
